User/group SID tips

S-1-1-0 means Everyone. Use it when the rule should apply to all users.

S-1-5-32-544 means local Administrators. Default AppLocker policies usually allow administrators to run everything.

For a specific AD group, get the SID with PowerShell:

Get-ADGroup "Group Name" | Select-Object Name,SID
